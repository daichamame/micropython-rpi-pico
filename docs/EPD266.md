# MicroPython 2.66inch E-Paper Driver for Raspberry Pi Pico  
**Waveshare 2.66inch E-Paper向け 実装ガイド**

このモジュールは、Raspberry Pi Pico と E-Paper を動作させるための軽量 MicroPython ドライバです。  

---

## 1. 対応デバイス
- Raspberry Pi Pico / Pico W  
- Waveshare 2.66inch E-Paper
- MicroPython  

---

## 2. モジュールの特徴
- 基本描画（線、アイコン）  
- 日本語テキスト描画対応（外部フォント方式 / 東雲フォントなど） 

---

## 3. インストール
```
/lib/daichamame_epd266.py
/fonts/（外部取得フォント）
```

---

## 4. 使い方（基本）

### 4.1 初期化例(東雲フォントのBDFファイルを読み込み例)
```python
import daichamame_epd266
import fontloader
fnt=fontloader.FontLoader("/font/shnm8x16r.bdf") # 東雲フォントのBDFファイルを読み込む
shnmfont=fnt.load_font()
epd2 = daichamame_epd266.Epd266(rotate=90,font_array=shnmfont,font_size=16)
epd2.init()
epd2.buffer_clear()
```

---

### 4.2 描画例
```python
epd2.clear() # 画面クリア
epd2.print(0,0,"E-Paper ｻﾝﾌﾟﾙ",1) # テキスト描画
epd2.line(50, 50,100,100) # 線を描く
epd2.display() # 画面更新
```

---

## 5. API リファレンス
- `init()	# 初期処理`
- `init_partial()	# 初期処理(反転表示されない)`
- `reset() # リセット`
- `clear() # 画面初期化`
- `buffer_clear() # 表示用配列を初期化`
- `display() # 画面更新`
- `set_rotate(rotate) # 画面回転(0,90,180,270)`
- `print(x,y,buf,ratio,color,invert,bold,vertical) # 文字描画`
- `line(sx,sy,ex,ey) # 線描画`
- `draw_icon(x,y,buf,ratio) # アイコン(32x32)描画`
